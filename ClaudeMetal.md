# meviy 절삭·선반 부품 설계 가이드라인

3D CAD 설계자를 위한 금속·수지 절삭 가공 설계 기준 문서입니다. MISUMI meviy 자동 견적 서비스가 형상을 인식하고 가공할 수 있도록, 모델링 단계에서 무엇을 지켜야 하는지 정리했습니다. 모든 수치와 규칙은 meviy 한국 기술정보 페이지를 출처로 하며, 본문 끝에 출처 링크를 첨부했습니다.

이 문서는 두 가지 서비스를 다룹니다. 절삭 플레이트는 3축 머시닝 센터로 직육면체 소재를 가공하는 서비스이고, 선반은 회전 대칭 형상을 가공하는 서비스입니다. meviy는 업로드된 3D 모델의 최외경 형상을 보고 절삭과 선반을 자동으로 구분합니다.

---

## 0. 설계 전 핵심 원칙 (먼저 읽으세요)

1. 제조는 업로드한 3D CAD 데이터를 기준으로 합니다. 도면 없이 3D 형상 자체가 가공 지시서가 되므로, 형상 요소를 빠짐없이 모델링해야 합니다.
2. meviy는 외형 치수와 홀 정보 외의 치수·공차를 처음에는 표시하지 않습니다. 공차가 필요한 부위는 설계자가 직접 화면에서 지정합니다.
3. 나사산은 3D 모델링으로 표현해도 인식되지 않습니다. 수나사·암나사는 meviy 화면에서 타입으로 지정합니다.
4. 가공 한계보다 얇거나 작은 형상은 자동 견적이 막히고 품질 동의 사항이 표시됩니다. 동의하지 않으면 담당자 견적으로 넘어갑니다.
5. 자동 견적이 불가능한 형상도 대부분 담당자 견적으로 가공할 수 있습니다. 형상을 단순화하면 자동 견적과 비용 절감에 유리합니다.

# 제1부. 절삭 플레이트 (3축 머시닝)

## 1.1 가공 방식과 기본 정밀도

절삭 플레이트는 직육면체 소재의 면에 대해 수직·평행 6방향으로 가공합니다. 직육면체에 대해 각도가 필요한 가공은 전용 지그가 필요하므로 서비스 대상에서 제외됩니다. 6면 방향에 대한 경사 가공은 3축 머시닝으로는 불가능합니다.

지시가 없을 때 적용되는 보통 허용공차와 마감 기준은 다음과 같습니다. 모두 원점을 가공 기준으로 적용하며, 원점은 임의 위치로 이동할 수 있습니다.

## 1.2 견적 가능 재질·표면처리

표면처리 종류에 따라 표면처리용 리프팅홀이 필요합니다. 리프팅홀 설계 규칙은 1.4를 참고하세요.

| 재질 분류 | 재질 | 표면처리 | 리프팅홀 |
|---|---|---|---|
| 스틸 | SM45C(S45C), SS400(SS41), SKD11 | 없음 | 불필요 |
| 스틸 | 동일 | 무전해 니켈도금, 크롬도금, 흑착색, 니켈도금, 경질 크롬도금 | 필요 |
| 알루미늄 | AL6061, AL5052, AL7075 | 없음 | 불필요 |
| 알루미늄 | 동일 | 백색·흑색 아노다이징, 무광 아노다이징, 경질 아노다이징 | 필요(동의 시 면제, 와이어 자국 가능) |
| 알루미늄 | 동일 | 크롬도금 | 필요 |
| SUS | SUS304, SUS316, SUS303 | 없음 | 불필요 |
| BRASS | 황동(C3604) | 없음 | 불필요 |

도금은 로트에 따라 색감 차이가 있을 수 있습니다. AL7075, PEEK, SKD11, BRASS는 담당자 견적으로 진행됩니다.

## 1.3 견적 가능 사이즈

재료 중량이 20kg를 초과하면 납기 연장 또는 서비스 대상 외가 될 수 있습니다. 중량은 X×Y×Z×비중×10⁻⁶ 로 계산하며 X≥Y≥Z 입니다.

| 재질 | 비중 | 재질 | 비중 |
|---|---|---|---|
| SM45C | 7.85 | SUS304 | 7.93 |
| SS400 | 7.87 | SUS316 | 7.98 |
| SKD11 | 7.87 | 황동(C3604) | 8.43 |
| A6061 | 2.7 | 폴리아세탈 | 1.41 |
| A5052 | 2.68 | MC나일론 | 1.16 |
| A7075 | 2.8 | PEEK | 1.32 |

### 두께(Z) 대응 범위

| 재질군 | 자동 견적 | 담당자 견적 | 대상 외 |
|---|---|---|---|
| 스틸·스테인리스 | 4 이상 70 이하 | 3~4 미만, 70 초과 120 이하 | 3 미만, 120 초과 |
| 알루미늄 | 4 이상 70 이하 | 3~4 미만, 70 초과 300 이하 | 3 미만, 300 초과 |

가로(X)·세로(Y)는 금속·스테인리스의 경우 X 2400 이하, Y 1200 이하까지 견적 가능하며, 10 미만이면 대상 외입니다. X 600 이상은 표준납기에 3일 이상 가산됩니다.

## 1.4 3D 모델링 핵심 규칙

### 엔드밀 궤적 R은 반드시 모델링

엔드밀이 그리는 궤적은 끝부분이나 반환점에 반드시 R이 생깁니다. 엔드밀 직경과 칼날 길이를 고려해 R 1.25 이상으로 모델링해야 합니다. R이 클수록 큰 직경의 엔드밀을 쓸 수 있어 가공 시간이 줄고 비용이 절감됩니다.

### 노즈 R은 모델링하지 않음

엔드밀 모서리의 극소 R인 노즈 R을 모델링하면 가공 방향 판정에 실패해 자동 견적이 불가능해집니다. 노즈 R은 생략합니다.

### 모따기 모델링 기준

C0.5를 초과하는 모따기는 반드시 모델링합니다. 모서리부가 핀각이거나 C0.5 이하이면 C0.1~0.5 이하로 자동 마감됩니다.

### 블라인드 홀 바닥면 차이

모델 바닥면이 평면이라도 가공 특성상 원뿔형으로 마감될 수 있고, 그 반대도 가능합니다. 이로 인해 홀이 관통해 다른 형상에 간섭·변형·팽창이 생길 수 있습니다. 평탄 바닥이 필수면 홀 정보 자유기재란에 "ΦOO, 깊이OO, 바닥면 플랫"으로 적어 담당자 견적을 의뢰합니다.

### 탭·정밀 홀 기초 홀 깊이 차이

기초 홀은 모델대로 가공되지 않을 수 있습니다. 유효 깊이가 "모델 깊이 − 드릴홈 깊이"를 초과하면 기초 홀이 관통해 간섭할 수 있습니다. 드릴홈 깊이 참고값은 피치×2.5+2mm 입니다.

### 리프팅홀 설계 (표면처리 시)

아노다이징·도금은 부품을 매달아 처리수조에 담그므로 리프팅홀이 필요합니다. 와이어 자국 동의 시에는 생략 가능합니다. 아래 형상을 2개 이상 조합합니다(동일 형상 2개도 가능).

| 형상 | 조건 |
|---|---|
| 관통 홀 | Φ3.5 이상 (정밀 홀 불가) |
| 타원 홀 | 폭 3.5 이상 (정밀 타원 홀 불가) |
| 클로즈 포켓 | 폭 3.5 이상 |
| 관통 탭 홀 | M5 이상 |

## 1.5 형상별 견적 가능 조건 (포켓·홀)

녹색 형상은 대상, 황색 형상은 대상 외가 되는 원인입니다. 황색 부분을 변경·삭제하면 대상 형상이 됩니다.

### 포켓

| 포켓 종류 | 조건 |
|---|---|
| 포켓 가공 | 엔드밀로 6방향에서 가공 가능한 형상만 대상 |
| 클로즈 포켓 (4면 둘러싸임) | 관통·비관통 무관, 모퉁이 4곳에 안쪽 R 필수(핀각 불가) |
| 클로즈 포켓 모따기 | C15 이하만 대상 (C15 초과 입구 모따기는 엔드밀 가공 불가) |
| 오픈 포켓 ① (4면 안 둘러싸임) | 안쪽 R 없어도 대상. 단 비관통은 가공 방향 R 필수 |
| 오픈 포켓 ② (바깥쪽 R 부착) | 엔드밀이 R에 수평으로 진입 못하는 형상은 대상 외 |

C15 이하 모따기에서 모따기 커터로 가공 가능한 형상이면 포켓 깊이에 제한이 없습니다.

### 홀·타원 홀

| 형상 | 조건 |
|---|---|
| 홀 | 다방향 가공 가능. 카운터보어 아랫단에 여러 홀·타원 홀 있어도 대상 |
| 2단 이상 홀 | 2단만 대상. 중간 단이 최소인 형상만 대상(중간 단 최대=보링은 대상 외) |
| 타원 홀 | 다방향·카운터보어 대상. 2단 이상도 대응(중간 단 최대 보링은 대상 외) |
| 홀+모따기 | 입구 모따기 C15 이하만 대상. 입구 바깥쪽 R 있으면 대상 외 |
| 홀+R | 모퉁이 R이 0.5 초과면 대상 외. 곡면에 모따기·R 있으면 대상 외 |

## 1.6 홀 인식 조건과 종류

meviy는 비관통·관통 원주, 입구 모따기 포함 형상, 1단·2단 홀을 "홀"로 인식합니다. 인식된 홀은 스트레이트 홀, 정밀 홀, 탭 홀, 인서트 홀, 접시 홀로 전환할 수 있습니다.

### 홀 종류별 인식 기준

| 종류 | 인식·설정 조건 |
|---|---|
| 스트레이트 홀 | 초기 인식에서 탭·접시에 해당하지 않는 홀 |
| 정밀 홀 | 스트레이트 홀의 홀 직경 공차 타입을 변경 |
| 탭 홀 | 유저 설정의 기초 홀 직경과 모델 홀 직경이 일치 |
| 인서트 홀 | 알루미늄 재질에서 탭과 동일 로직, 재질 SUS304, M2~M12 |
| 접시 홀 | 각도 90°, d≥3.0mm, d≥4.0일 때 D>1.4d / d<4.0일 때 D>1.7d |

### 탭 홀 수동 할당 가능 범위

| 탭 | 모델 홀 직경(Φ) | 탭 | 모델 홀 직경(Φ) |
|---|---|---|---|
| M2 | 1.5~2 | M8 | 6.6~8 |
| M2.5 | 2~2.5 | M10 | 8.3~10 |
| M3 | 2.4~3 | M12 | 10.1~12 |
| M4 | 3.2~4 | M14 | 11.8~14 |
| M5 | 4.1~5 | M16 | 13.8~16 |
| M6 | 4.9~6 | | |

탭 인식 정밀도를 높이려면 각 CAD의 홀 명령어로 생성한 모델링을 권장합니다. 돌출컷으로 만든 홀은 로직과 차이가 생기면 스트레이트 홀로 인식될 수 있습니다. SolidWorks는 탭 표현 방식에 따라 SW(A)와 SW(B)로 구분되며, 사용 설정과 다르면 인식이 어긋납니다.

### 서비스 대상에서 제외되는 "그 밖의 홀"

선단이 가는 2단 홀, 90° 이외 입구 모따기, 테이퍼 홀, 경사 홀, 입구 필렛, 절삭 나사산. 이 형상이 존재하면 서비스 대상에서 제외됩니다.

### 카운터싱크 설정 주의

KS규격 선택 시 기본 각도가 92°로 인식되어 형상 인식에 실패할 확률이 높습니다. ISO 또는 JIS 규격을 선택해 90°로 인식시켜야 합니다.

## 1.7 두께 판정 로직 (얇은 두께 가공 한계)

가공 한계 미만의 얇은 부분은 파손·변형으로 공차 이탈이 발생합니다. 한계값이 중복되면 큰 값을 우선합니다. 한계 이하면 품질 동의 사항이 표시되고, 동의하지 않으면 자동 견적이 불가능합니다. 단위는 mm입니다.

| 형상과 인접 형상 사이 | 조건 | 가공 한계 |
|---|---|---|
| 스트레이트 홀 | Φ2~Φ5 / Φ5 초과 | 0.8 / 1.0 |
| 정밀 홀 (스틸·알루미늄·SUS·BRASS) | Φ2 이상 | 0.8 |
| 정밀 홀 (폴리아세탈·MC나일론) | Φ2 이상 | 1.5 |
| 탭 홀 | M2~M5 / M6~M10 / M12 이상 | 0.8 / 1.0 / 1.5 |
| 인서트 홀 | M2~M5 / M6~M10 / M12 | 2.0 / 3.1 / 3.9 |
| 접시 홀 | Φ2~Φ5 / Φ5 초과 | 0.8 / 1.0 |
| 카운터보어 홀 (스틸·알루미늄·SUS·BRASS) | Φ3~Φ6 / Φ6 초과 | 0.8 / 1.0 |
| 카운터보어 홀 (폴리아세탈·MC나일론) | Φ3 이상 | 1.0 |
| 타원 홀 직선부 및 그 외 | 전체 | 1.0 |
| 포켓 직선부 및 그 외 | 전체 | 1.0 |
| 홀 끝단 | 전체 | 2.0 |

자동 견적 가능한 가공 깊이는 공구에 따라 다릅니다. 아래는 최대 기준값이며 형상·재질에 따라 변동됩니다.

## 2.7 키홈 규격

JIS B 1301:1996과 일치하는 키홈은 자동으로 공차가 설정되며 화면에서 변경할 수 있습니다. 규격과 불일치하면 외경 키홈은 타원홀로 인식되어 끼워맞춤 공차 지정이 불가하고, 내경 키홈은 일반 공차로만 인식됩니다.

외경 키홈은 모서리 끝부분이 둥근 형상이어야 인식됩니다. 둥글지 않으면 끼워맞춤 공차를 지정할 수 없으므로, 공차가 필요하면 홀 폭에 맞는 원형으로 설계합니다. 비관통 내경 키홈은 자동 견적 대상 외입니다.

키 폭 b에 따른 허용차는 P9, N9, H9, Js9, D10 등을 지원하며 b=2~32까지 정의되어 있습니다.

## 2.9 선반 모델링 외관 차이 유의점

### 코너·모서리 마감

코너가 C/R0.5 미만인 모델은 C/R0.1~0.4 또는 실면취로 가공합니다. 원통 모서리부가 0.5 미만이면 공구 날끝 R0.4로 마감합니다(릴리프 홈 없는 경우). C0.5 이상, R0.5 이상으로 모델링하면 3D 데이터대로 가공합니다. "면취 불가"는 지정할 수 없습니다.

### 센터 홀

회전 시 중심 흔들림 방지를 위해 단면에 센터 홀을 사용합니다. "센터 홀 제거"를 선택할 수 있으나 가공 가능 형상 범위에 제한이 생깁니다. 내경이 비어 있으면 센터 홀 제거 선이 표시되지 않습니다.

### 홀 바닥면 드릴 선단

홀 바닥에는 드릴 선단 형상이 생깁니다. 패턴 A는 선단 형상 발생, B는 선단 형상에 바닥면 없음, C는 단부에도 선단 형상 발생, D는 단부 선단 형상에 바닥면 없음입니다.

### 원통 모서리부 지정

설정창에서 "모서리부 R0 지정"을 선택하면 R0.2~R0.4의 폭으로 가공됩니다. 단 코너 R0.5 이상 모델링이 있거나 릴리프 홈이 있으면 적용되지 않습니다. 릴리프 홈이 없으면 원통 모서리는 임의 형상으로 가공됩니다.

### 불완전 나사부 (릴리프)

나사부 길이 확보에 필요한 릴리프 형상입니다. 아래 조건을 만족하지 않으면 미스미 추천치로 가공합니다. 단위는 mm입니다.

| 항목 | 수나사 | 암나사 |
|---|---|---|
| 불완전 나사부 길이(릴리프 홈 폭) 하한 | 피치×2.0 | 피치×2.5+2 |
| 나사부 길이(수)/깊이(암) 하한 | 피치×2.0 | 피치×2.0 |
| C면 | 버 방지 면취 | 버 방지 면취 |
| 릴리프 홈 깊이 최소 | 피치×0.75 | 피치×0.75 |

릴리프 홈 깊이를 피치×0.75 이하로 설정하면 가공은 되지만 나사 자국이 남습니다.

### 단면 홈·포켓 모서리

핀각 또는 R0.5 미만 형상은 R0.1~0.5 이하로 마감되므로 모델링이 불필요하며 핀각으로 설계합니다. 포켓 측면 가공 R은 공구 직경에 의존하며 가능한 작은 직경 공구를 선정합니다.

### 슬릿 가공

슬릿 가공으로 부품 치수가 달라질 수 있어, 해당 형상은 슬릿 가공 전 측정값으로 보증합니다.

## 부록. 설계 체크리스트

절삭 플레이트:

- 엔드밀 궤적 종점·반환점에 R1.25 이상을 넣었는가
- 노즈 R을 모델링하지 않았는가
- C0.5 초과 모따기를 모델링했는가
- 클로즈 포켓 모퉁이 4곳에 안쪽 R이 있는가
- 표면처리 시 리프팅홀 형상을 2개 이상 조합했는가
- 두께 한계(스트레이트 홀 0.8/1.0 등) 이상을 확보했는가
- 탭·정밀 홀을 CAD 홀 명령어로 작성했는가
- 카운터싱크를 ISO/JIS 90°로 설정했는가
- V홈·경사·5축·용접 형상이 없는가

선반:

- 최외경이 원통(각도 합 140° 초과)인가
- 나사산을 화면에서 타입으로 지정했는가
- 외경 키홈 모서리를 둥근 형상으로 설계했는가
- 키홈을 JIS B 1301 규격에 맞췄는가
- 관용 테이퍼 나사 각도를 1°~2.5° 미만으로 했는가
- 불완전 나사부 릴리프 홈 깊이를 피치×0.75 이상으로 했는가
- 스냅링 홈을 권장 m·코너 R값으로 설계했는가
- 단면 홈·포켓 모서리를 핀각으로 두었는가(R0.5 미만 자동 마감)
